# Digital Tamers 2 cross-save script 
Linux bash script for transferring save data between Android and Linux.

## Supported systems
```console
chmod +x dt2save.sh 
./dt2save.sh
```
## Options
-u SYSTEM<br>
Upload save data from Android to PC.
-d SYSTEM<br>
Download save data from PC to Android.
-b SYSTEM<br>
Backup PC save data.
-h<br>
Display help and exit.
